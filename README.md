# Team Integra - UN SDG 17

### Instructions for development
On Linux:
```
pip install virtualenv
cd [path to local repo]
virtualenv venv
source venv/bin/activate
pip install Flask
python __init__.py
```
On Windows:
```
pip install virtualenv
cd [path to local repo]
virtualenv venv
venv\Scripts\activate
pip install Flask
python __init__.py
```

The above steps assume you have python installed on the host system.

Navigate to localhost:5000 to view.

To deactivate the virtual environment:
```
deactivate
```
